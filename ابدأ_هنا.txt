         🚀 تفعيل نظام التحكم في إيقاف/تشغيل النظام
═══════════════════════════════════════════════════════════════

✅ تم الانتهاء من جميع التعديلات!

الآن فقط تحتاج تشغيل 3 أوامر:

═══════════════════════════════════════════════════════════════

📌 الخطوة 1: أوقف السيرفر
─────────────────────────────────────────────────────────────
اضغط Ctrl+C في Terminal لإيقاف السيرفر

─────────────────────────────────────────────────────────────

📌 الخطوة 2: شغل هذه الأوامر بالترتيب
─────────────────────────────────────────────────────────────

npx prisma generate

npx prisma db push

node scripts/init-system-settings.js

npm run dev

─────────────────────────────────────────────────────────────

⚠️ إذا ظهر خطأ EPERM في prisma generate:
  1. أغلق VS Code تماماً
  2. افتح Terminal جديد
  3. شغل الأوامر مرة أخرى

═══════════════════════════════════════════════════════════════

🎯 بعد التشغيل:

1. سجل دخول بحساب المطور (developer@system.local)

2. في Sidebar (القائمة الجانبية) ستجد زر في الأسفل:
   
   🟢 "النظام مفعل" = كل شيء شغال
   🔴 "النظام معطل" = النظام واقف

3. اضغط على الزر للتبديل

═══════════════════════════════════════════════════════════════

✨ المميزات الجديدة:

✓ حساب المطور لا يتعطل أبداً
✓ زر تحكم سريع في Sidebar
✓ صفحة إيقاف محسنة بثيم الداشبورد
✓ عند إيقاف النظام:
  - المطور: يدخل عادي ✅
  - باقي المستخدمين: ممنوعين ❌

✓ حماية كاملة من التجاوز:
  - تغيير الرابط يدوياً → توجيه فوري للإيقاف
  - استخدام أزرار المتصفح → ممنوع
  - فحص دوري كل 3 ثواني
  - توجيه تلقائي عند التفعيل
