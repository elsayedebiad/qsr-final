📍 المكان الثاني: أعلى صفحة Dashboard

┌─────────────────────────────────────────┐
│  🔥 إعادة تعيين النظام [للاختبار]    │
│  ─────────────────────────────────────  │
│  حذف جميع البيانات التجريبية          │
│  والبدء من جديد بقاعدة بيانات نظيفة   │
│                                         │
│  🔄 سيتم حذف السير الذاتية، العقود... │
│                                         │
│          [⚠️ إعادة تعيين الآن ⚠️]       │  ← هنا!
└─────────────────────────────────────────┘

الموقع: بطاقة كبيرة في أعلى صفحة Dashboard الرئيسية
التصميم: بطاقة ملونة بـgradient برتقالي/أحمر/وردي

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 كيفية الاستخدام:

1️⃣ افتح Dashboard
2️⃣ اختر أحد الزرين:
   • الزر في أسفل Sidebar (دائم الظهور)
   • أو البطاقة في أعلى الصفحة
3️⃣ اضغط على الزر
4️⃣ ستظهر نافذة تأكيد كبيرة
5️⃣ راجع قائمة البيانات التي سيتم حذفها
6️⃣ اضغط "نعم، احذف كل شيء" للتأكيد
7️⃣ انتظر... سيتم الحذف
8️⃣ إعادة تحميل تلقائية للصفحة

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🗑️ ما الذي سيُحذف؟

❌ جميع السير الذاتية (CVs)
❌ جميع العقود (Contracts)  
❌ جميع الحجوزات (Bookings)
❌ جميع الأنشطة (Activity Logs)
❌ جميع الإشعارات (Notifications)
❌ جميع البنرات (Banners)
❌ جميع الإعدادات (Settings)
❌ جميع المستخدمين (عدا المطور والمدير)

✅ سيبقى فقط:
✔️ حساب المطور (developer@system.local)
✔️ حساب المدير (admin@qsr.com)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ تم إصلاح Auto Refresh:

✅ السيرفر يعمل الآن: npm run dev
✅ عند أي تعديل في الكود → الصفحة تتحدث تلقائياً
✅ Hot Module Replacement (HMR) يعمل
✅ لا حاجة لتحديث يدوي
