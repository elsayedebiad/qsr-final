# قالب Excel للرفع الذكي مع عمود الحالة

## الأعمدة المطلوبة

### الأعمدة الأساسية
| اسم العمود | مطلوب | مثال | ملاحظات |
|-----------|-------|------|---------|
| الاسم الكامل | نعم | Fatima Ahmed | الاسم باللغة الإنجليزية |
| الاسم بالعربية | لا | فاطمة أحمد | الاسم باللغة العربية |
| الجنسية | نعم | فلبينية | الجنسية بالعربية |
| العمر | نعم | 28 | رقم |
| **الحالة** | **لا** | **جديد** | **الحالة الجديدة المضافة** |

### قيم عمود الحالة المقبولة

#### 1. جديد (NEW)
```
جديد | جديدة | NEW
```

#### 2. محجوز (BOOKED)
```
محجوز | محجوزة | حجز | BOOKED
```

#### 3. متعاقد (HIRED)
```
متعاقد | متعاقدة | عقد | تعاقد | HIRED
```

#### 4. مرفوض (REJECTED)
```
مرفوض | مرفوضة | رفض | REJECTED
```

#### 5. مرتجع (RETURNED)
```
مرتجع | مرتجعة | ارتجاع | راجع | RETURNED
```

#### 6. أرشيف (ARCHIVED)
```
أرشيف | ارشيف | مؤرشف | مؤرشفة | ARCHIVED
```

## مثال على الصفوف

### مثال 1: سير ذاتية جديدة
```
الاسم الكامل: Maria Santos
الاسم بالعربية: ماريا سانتوس
الجنسية: فلبينية
العمر: 30
الحالة: جديد
```

### مثال 2: سيرة ذاتية متعاقدة
```
الاسم الكامل: Sarah Ahmed
الاسم بالعربية: سارة أحمد
الجنسية: إندونيسية
العمر: 28
الحالة: متعاقد
```

### مثال 3: سيرة ذاتية في الأرشيف
```
الاسم الكامل: Noor Fatima
الاسم بالعربية: نور فاطمة
الجنسية: بنغالية
العمر: 35
الحالة: أرشيف
```

## ملاحظات مهمة

1. **عمود الحالة اختياري**: إذا لم تضف عمود الحالة، ستكون جميع السير الذاتية بحالة "جديد" افتراضيًا
2. **يمكن استخدام العربية أو الإنجليزية**: النظام يفهم كلا اللغتين
3. **لا يهم حجم الأحرف**: "جديد" و "جَديد" و "JADED" كلها مقبولة
4. **القيم غير المعروفة**: إذا أدخلت قيمة غير معروفة، ستكون الحالة "جديد" افتراضيًا

## تحميل القالب

يمكنك تحميل قالب Excel جاهز من لوحة التحكم:
1. انتقل إلى صفحة الرفع الذكي
2. اضغط على "تحميل القالب"
3. املأ البيانات
4. ارفع الملف
