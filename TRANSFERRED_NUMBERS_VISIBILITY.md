# ✅ إصلاح: الأرقام المحولة تبقى ظاهرة في القائمة

## التغييرات المطبقة:

### 1️⃣ تغيير التسميات من "معطل" إلى "محول"

**قبل:** ❌
```
🔴 معطل
زر: [عرض المعطلة]
```

**بعد:** ✅
```
🔴 محول
زر: [عرض المحولة]
```

---

### 2️⃣ الأرقام المحولة تبقى في القائمة

**القديم:** ❌
- الرقم المحول (المنتهي) يختفي من قائمة الشخص الأصلي

**الجديد:** ✅
- الرقم يبقى ظاهراً (مع blur + رسالة "تم سحب الرقم")

---

## 📋 كيف يظهر الرقم المحول:

### **للشخص الأصلي (تم سحب الرقم منه):**

```
┌─────────────────────────────────────┐
│ الرقم: ████████ (blur effect)      │
│        تم حجب الرقم                 │
│ 🔴 محول                            │
│ ⚠️ تم سحب الرقم                    │
├─────────────────────────────────────┤
│ الصفحة: sales1                     │
│ (لا معلومات عن التحويل)            │
└─────────────────────────────────────┘
```

### **للأدمن:**

```
┌─────────────────────────────────────┐
│ الرقم: 0501234567 (واضح!)         │
│ 🔴 محول                            │
├─────────────────────────────────────┤
│ الصفحة: sales1                     │
│         → أحمد                      │
│ [🔄 إعادة تفعيل]                   │
└─────────────────────────────────────┘
```

### **للمستلم (لم يتواصل):**

```
┌─────────────────────────────────────┐
│ الرقم: ████████ (blur effect)      │
│        تم حجب الرقم                 │
│ 🔴 محول                            │
│ ⚠️ تم سحب الرقم                    │
├─────────────────────────────────────┤
│ الصفحة: sales1                     │
│         ← محول إليك                │
└─────────────────────────────────────┘
```

## 🔧 التعديلات التقنية:

### 1. Badge "محول" بدلاً من "معطل":
```tsx
{item.isExpired && (
  <span className="bg-red-100 text-red-800">
    <AlertCircle /> محول
  </span>
)}
```

### 2. زر "عرض المحولة":
```tsx
{showExpired ? 'إخفاء المحولة' : 'عرض المحولة'}
```

### 3. منطق الفلترة في API:
```typescript
// الأدمن: يختار عرض المحولة أو النشطة
if (user.role === 'ADMIN' && showExpired) {
  where.isExpired = true
} else if (user.role === 'ADMIN') {
  where.isExpired = false
}
// المستخدم العادي: يرى الكل (محولة + نشطة)
// لا فلتر على isExpired
```

## 🎯 الفوائد:

### ✅ الشفافية:
- الشخص الأصلي يعرف أن رقمه تم سحبه (مع blur)
- لا يختفي الرقم فجأة من القائمة

### ✅ السجل الكامل:
- جميع الأرقام (النشطة والمحولة) تبقى مرئية
- سهولة التتبع والمراجعة

### ✅ الخصوصية محفوظة:
- الشخص الأصلي لا يرى **لمن** تم التحويل
- فقط رسالة "تم سحب الرقم"

### ✅ للأدمن:
- يمكنه رؤية كل التفاصيل
- يمكنه التبديل بين المحولة والنشطة
- زر إعادة التفعيل متاح

| ظهور الرقم | يختفي ❌ | يبقى مع blur ✅ |
| معلومات التحويل | ظاهرة للجميع | مخفية عن الأصلي ✅ |
