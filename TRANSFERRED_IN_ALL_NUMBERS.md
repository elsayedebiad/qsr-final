# ✅ الأرقام المحولة تظهر في "جميع الأرقام" للأدمن

## المشكلة التي تم حلها:

**قبل:** ❌
```
الأدمن يختار "جميع الأرقام"
↓
يرى فقط الأرقام النشطة (المحولة مخفية!)
```

**بعد:** ✅
```
الأدمن يختار "جميع الأرقام"
↓
يرى كل شيء (نشطة ✅ + محولة ✅)
```

---

## 🎯 طرق عرض المحولة للأدمن:

### 1️⃣ **من dropdown الصفحات**:
```
[جميع الصفحات ▼]
├─ جميع الصفحات     ← يعرض الكل (نشطة + محولة)
├─ sales1
├─ sales2
├─ 🔴 المحولة        ← يعرض المحولة فقط
└─ نقل الخدمات
```

### 2️⃣ **من زر "عرض المحولة"**:
```
[عرض المحولة]  ← الضغط عليه يعرض المحولة فقط
```

### 3️⃣ **من Card "المحولة"**:
```
┌─────────┐
│🔴محولة │  ← الضغط عليها يعرض المحولة فقط
│   15   │
└─────────┘
```

## 🔧 التعديلات التقنية:

### 1. **إضافة "المحولة" في القائمة** (`page.tsx`):
```typescript
const salesPages = [
  { id: 'ALL', name: 'جميع الصفحات' },
  { id: 'sales1', name: 'صفحة المبيعات 1' },
  // ...
  ...(isAdmin ? [{ id: 'TRANSFERRED', name: '🔴 المحولة' }] : []),
  { id: 'transfer-services', name: 'نقل الخدمات' }
]
```

### 2. **معالجة TRANSFERRED في API** (`list/route.ts`):
```typescript
if (salesPageId === 'TRANSFERRED') {
  // عرض المحولة فقط
  where.isTransferred = true
  where.isExpired = true
}
```

### 3. **إزالة فلتر isExpired للأدمن** (`list/route.ts`):
```typescript
// القديم ❌:
if (user.role === 'ADMIN') {
  where.isExpired = false  // يخفي المحولة!
}

// الجديد ✅:
if (user.role === 'ADMIN' && showExpired === 'true') {
  where.isExpired = true  // فقط عند الضغط على "عرض المحولة"
}
// وإلا: لا فلتر (يرى الكل)
```

## ✨ الفوائد:

### ✅ **رؤية شاملة**:
- الأدمن يرى كل شيء في مكان واحد
- لا حاجة للتنقل بين "النشطة" و "المحولة"

### ✅ **مرونة في العرض**:
- يمكن رؤية الكل (ALL)
- يمكن رؤية المحولة فقط (TRANSFERRED أو زر)
- يمكن رؤية صفحة معينة مع محولاتها

### ✅ **تمييز واضح**:
- الأرقام المحولة لها badge "🔴 محول"
- سهل التفريق بينها وبين النشطة

### ✅ **3 طرق للوصول**:
1. من dropdown القائمة
2. من زر "عرض المحولة"
3. من Card المحولة
